Συχνές Ερωτήσεις

  1. Πώς μπορώ να κάνω εγγραφή ;


    2. Πατάτε το κουμπί Login και εμφανίζεται το πλαίσιο εισόδου και πατάτε Register και οδηγείστε στη φόρμα εισαγωγής στοιχείων.

  2. Πώς χρησιμοποιούνται τα προσωπικά μου στοιχεία ;


    3. Όλες οι πληροφορίες βρίσκονται στο Πολιτική Χρήσης.

  3. Πώς γίνεται η επιβεβαίωση του λογαριασμού μου ;


    4. Θα ζητηθεί μία μόνο φορά επιβεβαίωση μέσω τηλεφώνου στην πρώτη σας παραγγελία.

  4. Πώς μπορώ να κάνω παραγγελία ;


    5. Κατεβαίνετε στην κεντρική σελίδα στο "Φτιάξε την κρέπα σου" και διαλέγετε τα υλικά που θέλετε. Αφού τα επιλέξετε πατάτε το "καταχώρηση κρέπας" ( Αν αλλάξατε γνώμη πατάτε ακύρωση κρέπας ).Αφού τελειώσετε με την παραγγελία συμπληρώνετε τα στοιχεία σας ( Αν είστε ήδη εγεγγραμένος και συνδεδεμένος θα συμπληρωθούν αυτόματα) και πατάτε αποστολή παραγγελίας.

  5. Τα υλικά που χρησιμοποιούνται είναι φρέσκα ή κατεψυγμένα ;


    6. Τα υλικά που χρησιμοποιούμε είναι φρέσκα εκτός από τα θαλασσινά είδη ( καλαμαράκια ).

  6. Ποιά είναι η ελάχιστη παραγγελία ;


    7. Η ελάχιστη παραγγελία είναι 3,5 €.

  7. Πόση ώρα κάνει να φτάσει η παραγγελία ;


    8. Σε ώρες αιχμής κυμαίνεται από 20 έως 30 λεπτά.

  8. Σε πόσες περιοχές των Σερρών παραδίδεται παραγγελίες ;


    9. Παραδίδουμε μόνο μέσα στην πόλη των Σερρών.
